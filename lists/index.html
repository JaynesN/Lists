What to use to run mods:

Forge
The oldest and most popular version, has the most capabilities but takes more computer power, It can also be slightly annoying to work with, for both modders and coders.
Fabric
Much easier to run, but has less mods due to the fact that it has less options for things to happen ingame.
Quilt
Fairly New and very lightweight, it is actually built off of fabric, meaning it can run most mods built for fabric.
